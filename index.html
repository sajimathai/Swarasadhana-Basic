📱
ഫോണിൽ (iPhone / Android)
  1. Android (Chrome): വലത് മുകളിലെ ⋮ മെനു → "Add to Home screen" / "ഇൻസ്റ്റാൾ ആപ്പ്" തിരഞ്ഞെടുക്കുക.
  2. iPhone (Safari): താഴെയുള്ള ഷെയർ ⬆️ ബട്ടൺ → "Add to Home Screen" തിരഞ്ഞെടുക്കുക.
  3. ഹോം സ്ക്രീനിൽ വരുന്ന ഐക്കണിൽ ടാപ്പ് ചെയ്താൽ ആപ്പ് പോലെ തുറക്കും.
💻
Windows / Mac ഡെസ്ക്ടോപ്പിൽ
  1. Chrome / Edge ബ്രൗസറിൽ ഈ പേജ് തുറക്കുക.
  2. വിലാസബാറിന്റെ വലത്തെ ഇൻസ്റ്റാൾ ഐക്കൺ (⊕ / 🖥️) ക്ലിക്ക് ചെയ്യുക, അല്ലെങ്കിൽ ⋮ മെനു → "Install".
  3. ഡെസ്ക്ടോപ്പിൽ ഒരു സ്വന്തം വിൻഡോയായി ആപ്പ് തുറക്കും.
💡 ഇൻറർനെറ്റ് ഇല്ലാതെയും ഉപയോഗിക്കാൻ, ഡൗൺലോഡ് ചെയ്ത സ്റ്റാൻഡ്എലോൺ ഫയൽ (Swarasadhana-standalone.html) ഇരട്ട-ക്ലിക്ക് ചെയ്ത് തുറക്കാം. മൈക്ക് റെക്കോർഡിംഗ് അങ്ങനെ തുറന്നാൽ പ്രവർത്തിക്കും.
